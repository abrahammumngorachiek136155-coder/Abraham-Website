HTML Text Formatting Examples

This text is normal.

This is a bold text using the `<b>` tag.
This text has strong importance using the `<strong>` tag.

This is an italic text using the `<i>` tag.
This text is emphasized using the `<em>` tag (which also typically renders as italic).

This text is marked or highlighted using the `<mark>` tag.

This is some smaller text using the `<small>` tag.

This text has been deleted using the `<del>` tag, and this text has been inserted using the `<ins>` tag.

This is subscripted text (e.g., in a chemical formula like H2O) using the `<sub>` tag.

This is superscripted text (e.g., in a mathematical expression like E = mc2) using the `<sup>` tag.

This text includes an underlined section using the non-standard but often rendered `<u>` tag.

This uses the `<code>` tag to display a snippet of computer code, which typically uses a monospace font.
